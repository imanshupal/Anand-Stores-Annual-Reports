 📊 Excel Store Sales Dashboard

An interactive **Store Sales Dashboard** built using Microsoft Excel to analyze sales performance, customer demographics, order status, and sales channels.

📌 Project Overview

This dashboard converts raw sales data into meaningful business insights using Pivot Tables, Pivot Charts, and Slicers.

🚀 Dashboard Features

- 📅 Monthly Filter using Slicers
- 📈 Orders vs Sales Analysis
- 👨‍💼 Men vs Women Sales Distribution
- 📦 Order Status Analysis
- 🏆 Top 5 States by Sales
- 👥 Age vs Gender Analysis
- 🛒 Sales by Sales Channels
- 📊 Interactive Charts
- 🎨 Clean & Professional Dashboard Design

🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Analysis
- Dashboard Design
